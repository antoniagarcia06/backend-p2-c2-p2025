 Agenda API – Django REST Framework + JWT

API RESTful para la gestión de contactos, desarrollada con **Django**, **Django REST Framework** y **SimpleJWT** para autenticación mediante **tokens Bearer**.  
Incluye endpoints protegidos, consumo desde **Thunder Client**, políticas de acceso y buenas prácticas de estructura y seguridad.
Características principales

CRUD completo de Contactos  
Autenticación con JSON Web Token (JWT)  
 Roles y permisos: IsAuthenticated, AllowAny  
Protección CSRF (formularios HTML / Django clásico)  
Consumo de la API desde Thunder Client / Postman  
 Implementación de CORS para clientes externos
